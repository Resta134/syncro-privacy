# Kebijakan Privasi Aplikasi Syncro

Terakhir diperbarui: 12 Juli 2026

Aplikasi Syncro ("kami") berkomitmen untuk melindungi dan menghormati privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat Anda menggunakan aplikasi mobile Syncro.

## 1. Informasi yang Kami Kumpulkan
Kami dapat mengumpulkan informasi berikut untuk mendukung fitur utama aplikasi:
- **Informasi Akun:** Nama, alamat email, dan kredensial login saat Anda mendaftar atau masuk ke aplikasi.
- **Data Verifikasi:** Data enkripsi atau otentikasi yang diperlukan untuk validasi sistem acara Anda.
- **Informasi Teknis:** Informasi perangkat, versi sistem operasi, dan log performa aplikasi untuk perbaikan bug.

## 2. Penggunaan Informasi
Kami menggunakan data yang dikumpulkan untuk:
- Menyediakan, mengoperasikan, dan menjaga fungsi aplikasi Syncro.
- Memproses otentikasi dan manajemen akses acara secara aman.
- Meningkatkan performa aplikasi dan memberikan pembaruan sistem.

## 3. Keamanan Data
Keamanan data Anda adalah prioritas kami. Kami menggunakan metode enkripsi yang aman saat data ditransmisikan (HTTPS) dan memastikan penyimpanan data dilakukan dengan standar keamanan yang baik pada server database kami.

## 4. Berbagi Data dengan Pihak Ketiga
Kami tidak menjual, memperdagangkan, atau menyewakan informasi pribadi pengguna kepada pihak ketiga. Data Anda hanya digunakan di dalam ekosistem aplikasi Syncro untuk kebutuhan operasional acara Anda.

## 5. Perubahan pada Kebijakan Privasi Ini
Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan berlaku segera setelah kebijakan baru diunggah di halaman ini.
